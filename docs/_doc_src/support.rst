Support
=======

Support for users is available through `GitHub Discussions
<https://github.com/ThibHlln/smartpy/discussions>`_. Use this support
page to find answers to previously asked questions, to ask questions
of your own, and to suggest ideas to improve `smartpy`.

To report bugs, please use `GitHub Issues <https://github.com/ThibHlln/smartpy/issues>`_.
