Tutorial
========

This tutorial showcases an example of the use of the API of `smartpy` for
running a :doc:`single experiment <tutorial/single_experiment>` (i.e. one
simulation with one parameter set) and a :doc:`Monte Carlo experiment
<tutorial/montecarlo_experiment>` (i.e. several simulations with a
sample of parameter sets):

.. toctree::
   :maxdepth: 1

   tutorial/single_experiment
   tutorial/montecarlo_experiment

It uses the input files available in the folder *examples/in/ExampleDaily*.
The output files that should be generated following this tutorial are made
available for comparison purposes in *examples/out/ExampleDaily*. Although,
given the stochastic nature of the Monte Carlo experiment, do not expect
the exact same outputs.
